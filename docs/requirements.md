# 项目需求文档 — Touch Tang 🐾

## 项目简介
做一个可爱的互动网页，页面中间有一只黑猫，猫的眼神会跟着鼠标移动。

## 用户能看到什么

1. **可爱的外观** — 米色/奶油色的背景，圆润可爱的字体，暖萌风格
2. **一只黑猫图片** — 猫咪在页面中间，眼神会跟随鼠标方向变化
3. **语言切换** — 页面顶部有按钮，可以切换 中文 / English / 한국어
4. **背景音乐播放器** — 页面底部有一个音乐播放器（音乐文件后续加入）
5. **双击摸摸** — 双击猫咪，猫咪会有被摸的反应图片 + 猫叫声

## 猫咪眼神方向说明

有 16 张方向图片（00.png ~ 15.png），从正下方开始，顺时针转一圈：

| 图片 | 方向 |
|------|------|
| 00.png | 正下方 ↓ |
| 01.png | 左下偏下 |
| 02.png | 左下 |
| 03.png | 左偏下 |
| 04.png | 正左 ← |
| 05.png | 左偏上 |
| 06.png | 左上 |
| 07.png | 上偏左 |
| 08.png | 正上方 ↑ |
| 09.png | 上偏右 |
| 10.png | 右上 |
| 11.png | 右偏上 |
| 12.png | 正右 → |
| 13.png | 右偏下 |
| 14.png | 右下 |
| 15.png | 下偏右 |

当鼠标靠近猫咪脸部中心（距离小于约 80 像素），显示 x.png（眼神看中间）。

## 图片资源

所有图片在 `tang_pic/` 文件夹中，共 18 张：
- `x.png` — 初始状态，眼神看中间
- `00.png` ~ `15.png` — 16 个方向
- `touch.png` — 被摸时的反应

## 音频资源

- `cat.m4a` — 猫叫声（双击时播放）
- 背景音乐文件 — 用户后续提供

## 支持的浏览器
- Chrome、Safari、Firefox 桌面版
- 手机浏览器（简单适配，图片缩小居中）

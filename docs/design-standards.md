# 设计规范文档 — Touch Tang 🐾

## 整体风格

- **主题**：可爱、柔和、治愈系
- **不要科技感**：避免冷色调、直角、深色背景
- **参考风格**：宠物互动网页、猫咪咖啡馆网站

## 色彩方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 页面背景 | `#fef9ef` 或 `#fff8e7` | 温暖奶油色 |
| 卡片/容器背景 | `#ffffff` 或 `#fffdf5` | 带一点暖调的白色 |
| 主文字 | `#5d4037` | 温暖的深棕色（不用纯黑） |
| 副文字 | `#8d6e63` | 柔和棕色 |
| 强调色 | `#f8bbd0` 或 `#ffcdd2` | 淡粉色（按钮、装饰） |
| 语言按钮选中 | `#ffab91` | 柔和珊瑚色 |

## 字体

- **主字体**：[Baloo 2](https://fonts.google.com/specimen/Baloo+2) — 可爱圆润，支持拉丁字母
- **中文字体**：系统默认圆体（PingFang SC / Microsoft YaHei）
- **韩文字体**：系统默认（Apple SD Gothic Neo / Malgun Gothic）
- **回退顺序**：`'Baloo 2', 'Nunito', 'PingFang SC', 'Microsoft YaHei', sans-serif`
- 标题字号约 2.5rem，有弹跳动画
- 副标题字号约 1.2rem

## 猫咪图片显示规则

- 桌面端显示宽度：约 500px（等比缩放，高度自动）
- 图片使用 `width` + `height` 固定，避免切换时尺寸变化
- 使用 `object-fit: contain` 确保不变形
- 图片容器加柔和阴影：`box-shadow: 0 8px 32px rgba(0,0,0,0.1)`
- 容器圆角：`border-radius: 20px`

## 语言切换按钮

- 位置：页面顶部居中
- 样式：胶囊形（pill shape），柔和颜色
- 3 个按钮并排，间距约 8px
- 当前选中的语言高亮显示

## 音乐播放器

- 位置：页面底部
- 使用浏览器原生 `<audio controls>` 但通过 CSS 美化
- 圆角、柔和边框、融入整体风格
- 桌面端宽度约 300px，居中
- 不自动播放（浏览器策略限制）

## 装饰元素

- **爪印 🐾**：2-3 个，绝对定位，缓慢浮动动画
- **爱心 ❤️**：2-3 个，从底部向上漂浮的 CSS 动画
- 装饰要**少量、精致**，不要铺满屏幕
- 所有装饰透明度约 0.3-0.5，不能喧宾夺主

## 响应式设计

- 桌面端（>768px）：完整布局，猫咪图片约 500px 宽
- 手机端（≤768px）：猫咪图片缩小至约 280px 宽，居中对齐
- 字体在手机上适当缩小
- 语言按钮在手机上保持可点击的大小（至少 36px 高度）
